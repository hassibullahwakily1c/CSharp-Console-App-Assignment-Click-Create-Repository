# Employee Interface Assignment

This console application demonstrates the use of an interface in C#.

## What the program does

- Creates an interface called `IQuittable`
- Defines a `Quit()` method inside the interface
- Has the `Employee` class implement that interface
- Uses polymorphism by assigning an `Employee` object to an `IQuittable` reference
- Calls the `Quit()` method through the interface

## Files included

- `Program.cs`
- `Employee.cs`
- `IQuittable.cs`

## Notes

I added comments throughout the code so each part is easy to follow and understand.
